#-MaskPhish is a simple script to hide phishing URL under a normal looking URL(google.com or facebook.com).


#-Installing (Tested on Kali Linux, Termux & Ubuntu):

git clone https://github.com/jaykali/maskphish
cd maskphish
bash maskphish.sh
